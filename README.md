# ComplyLight Core Common Libraries and Utilities

The NPM package provides common shared libraries and utilities functions for writting applications integrated with the [ComplyLight](https://www.complylight.com) stack, originally based on the Open Source [ASU SHARES](https://www.asushares.com) stack. It can be used in both backend applications such as Node and Express, as well as frondend applications such as Angular and React.


## Project Integration

`npm i -S @complylight/core` to install and save into your project's dependencies list


## Development

```shell
npm i # to install project dependencies
npm test-watch # to rerun test cases upon changes
```

## Publication

```shell
npm run build
npm publish --access public
```
